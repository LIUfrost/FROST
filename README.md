# 一.网站资源
## 1.综合资源库
- [fmhy](https://fmhy.net):此网站整合多种资源，包括多种AI大模型，开发者网站，语言学习网站等等，可以说应有尽有了。
- [Z-Library](https://zh.z-library.sk):全球最大免费书籍网站之一，支持免费下载。
## 2.CS资源库
- [CS自学指南](https://csdiy.wiki)：来自北大信科大佬的计算机自学指南，本人自学大多跟着它的指导，最推荐的自学网站。
- [Github](https://github.com):世界上最大的同性交友平台（雾），既然已经看到这了，含金量不必多说。
- [Stack Overflow](https://stackoverflow.com):程序员的知乎，大佬云集之地（但IP纯度似乎要求过高？登了好久没登上去...）
- [hellogithub](https://hellogithub.com):定期分享github优质项目，并有精心分类和详细介绍（另有同名公众号）
- [hugging face](https://huggingface.co):著名人工智能模型与数据集的托管与分享平台，支持下载和在线运行AI模型（deepseek本尊便在此开源）
- [Advent to Code](https://adventofcode.com):趣味编程解密网站，每天全球几十万程序员同时等新题。很多大厂工程师、算法高手、大学教授也在玩，是程序员节日一样的活动。
- [Coursera](https://coursera.org):全球最大在线学习网站（世界版MOOC），众多名校，大厂在此开设课程（另有同名软件，已放仓库），跟着CS自学指南自会找到课程。
- [Leetcode](https://leetcode.com):力扣，“老少皆宜”的编程刷题网站，无论是数据结构/算法初学者，还是准备面试的程序员，都可以获得很大帮助（同名软件自行应用商店下载）
